# 🔐 Blowfish Encryption Demo in Python

A simple Python program demonstrating **Blowfish encryption and decryption** using the **PyCryptodome** library.

---

## 📖 Overview

This project demonstrates the fundamentals of Blowfish symmetric encryption. It accepts plaintext from the user, encrypts it, and then decrypts it back to verify correctness.

> **Note:** This implementation uses **ECB (Electronic Codebook)** mode for educational purposes. ECB mode is not recommended for production environments because it reveals patterns in encrypted data.

---

## ✨ Features

- Blowfish Encryption
- Blowfish Decryption
- PKCS#7 Padding
- Command-Line Interface
- Beginner-friendly implementation

---

## 🛠 Technologies Used

- Python 3
- PyCryptodome

---

## 🚀 Installation

Install the required package:

```bash
pip install pycryptodome
```

Run the program:

```bash
python blowfish_demo.py
```
